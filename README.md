# Operacional_LogRev
Dashboard executivo para GitHub Pages.

## Visuais
- Baixados: barras horizontais empilhadas.
- Estoque de terceiras: treemap proporcional.
- Estoque de lojas: mapa de calor em cartões (Top 14).
- Internalizados: barras horizontais empilhadas.
- Tecnologia: barras verticais com o volume internalizado na data mais recente da base.

Substitua os três CSVs da pasta `data` pelos arquivos reais e publique a raiz na branch `main` pelo GitHub Pages.


## Estrutura simplificada
- `index.html`, `style.css` e `app.js` ficam diretamente na raiz.
- Apenas os CSVs permanecem dentro da pasta `data`.
